# Python-Login-GUI-Customtkinter

Simple Login Screen using customtkiner.
This Project was made possible with TkinterDesigner.

The GUI covers login, Create Account, Terms and Conditions, and Forget Password pages.

The project itself is proof of concept and contains logic errors and slow algorithms.

# Requirements
- Customtkinter
- Tkinter

# How to Run 
The main program is gui.py

Run using:
- python3 gui.py
